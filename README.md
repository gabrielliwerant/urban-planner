# Urban Planner

Allows creation of a schedule of activities for up to 31 days.

## Dependencies

- node v10.13.0

## Running Project in Development

  - `npm install`
  - `npm run start`
  - Navigate to `http://localhost:8080`

## Production

  - `npm run build` to bundle js src
  - Run `/dist/index.html` to load production app
